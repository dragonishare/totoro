```
yarn build
```


```
yarn lint
```

### Customize configuration

### 目录结构

* packages: 组件源码
* website: 原src目录，改为组件文档官网，展示示例
* website/docs: 组件文档和示例代码 markdown格式
* src: 组件公用工具函数
* src/theme: 组件样式文件夹（组件所有样式统一放这里）
* src/index.ts: 组件统一导出入口文件（整体导出，全量包）
